# Task Management API 📝

## Description
A RESTful API for managing tasks, built using ASP.NET Core.

## Features
- Create, update, delete tasks
- Structured with Controllers, Services, DTOs
- Database integration using Entity Framework

## Tech Stack
- C#
- ASP.NET Core
- Entity Framework Core
- SQLite

## What I learned
- Building REST APIs
- Clean architecture (Controller-Service pattern)
- Working with databases and migrations
