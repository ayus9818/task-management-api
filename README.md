# Task Management API

## Description
A RESTful API for managing tasks, built using ASP.NET Core. The project follows a clean architecture approach with separation of concerns using controllers, services, and data transfer objects (DTOs).

## Features
- Create, update, delete, and retrieve tasks
- Structured API with controller-service pattern
- Database integration using Entity Framework Core
- Clean and maintainable code structure

## Tech Stack
- C#
- ASP.NET Core
- Entity Framework Core
- SQLite

## Project Structure
- `Controllers/` – Handles API endpoints
- `Services/` – Business logic layer
- `Models/` – Data models
- `DTOs/` – Data transfer objects

## What I Learned
- Building RESTful APIs with ASP.NET Core
- Implementing clean architecture (Controller-Service pattern)
- Working with databases using Entity Framework Core
- Structuring scalable backend applications
- Handling CRUD operations efficiently
